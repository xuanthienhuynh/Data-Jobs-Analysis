# 📊 Power BI Portfolio: Data Jobs Market Analysis

Welcome to my Data Analyst portfolio repository! This project focuses on analyzing the data job market (Data Analysts, Data Scientists, Data Engineers, etc.) to uncover trends in top skills, median salaries, and job distributions. 

To demonstrate my learning progression and problem-solving mindset, this project was developed and documented in two distinct phases.

## 🌟 Project Versions

### 🏆 [Version 2.0 - Interactive Dashboard (Final Version)](./DataJob_ver2/README.md)
This is the final, highly polished version optimized for User Experience (UI/UX) and advanced analytical capabilities.
- **Highlights:** Implementation of **Field Parameters** for dynamic axis/metric switching, advanced **DAX** (`CALCULATE`, `ALL`) for complex benchmarking, and a professional Dark Theme UI.
- 👉 **[Click here to explore Version 2.0 details & Insights](./DataJob_ver2/README.md)**

### 📂 [Version 1.0 - Foundation & Data Modeling](./DataJob_ver1/README.md)
The foundational phase focusing on backend data engineering.
- **Highlights:** Extensive Data Wrangling using **Power Query**, error handling, and constructing a highly efficient **Star Schema** (Fact & Dimension tables).
- 👉 **[Click here to explore Version 1.0 details](./DataJob_ver1/README.md)**
