# 📊 Power BI Portfolio: Data Jobs Market Analysis

Chào mừng bạn đến với kho lưu trữ dự án Phân tích thị trường việc làm ngành Dữ liệu (Data Jobs). Dự án này được phát triển qua 2 giai đoạn để thể hiện sự tiến bộ trong tư duy phân tích và kỹ năng sử dụng Power BI.

## 🌟 Các phiên bản dự án (Project Versions)

### 🏆 [Version 2.0 - Interactive Dashboard (Bản hoàn chỉnh)](./DataJob_ver2/README.md)
Đây là phiên bản nâng cấp toàn diện, được thiết kế tối ưu cho trải nghiệm người dùng (UI/UX).
- **Tính năng nổi bật:** Tích hợp Field Parameters để thay đổi linh hoạt các góc nhìn (Metrics & Axis) mà không làm rườm rà giao diện.
- **Kỹ thuật:** Sử dụng DAX nâng cao (`CALCULATE`, `ALL`) và thiết kế Dark Theme chuẩn nguyên tắc thị giác (Z-Pattern).
- 👉 **[Xem chi tiết Version 2.0 tại đây](./DataJob_ver2/README.md)**

### 📂 [Version 1.0 - Foundation & Data Modeling](./DataJob_ver1/README.md)
Phiên bản nền tảng tập trung vào kỹ thuật xử lý dữ liệu dưới nền tảng.
- **Tính năng nổi bật:** Dọn dẹp dữ liệu thô bằng Power Query, xây dựng cấu trúc Star Schema (Fact & Dimension tables).
- 👉 **[Xem chi tiết Version 1.0 tại đây](./DataJob_ver1/README.md)**
